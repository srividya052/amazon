# Amazon Homepage Clone

A beginner frontend project inspired by the Amazon homepage. Built using HTML and CSS to practice webpage structure, layout design, styling, and responsive development.

## Features

* Amazon-inspired homepage design
* Navigation bar
* Product sections
* Responsive layout
* Clean HTML structure
* CSS styling and positioning

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created to strengthen frontend development fundamentals, including page structure, styling, layout design, and responsive web design.
